# 퍼징 캠페인 명령 셋

현재 구현과 문서 기준에서 실제로 추천 가능한 퍼징 명령만 모아둔 문서다.

- 기본 우선순위는 `real-ue-direct` 와 실기기 기준이다.
- 아래 명령은 `--impi` 를 일부러 넣지 않았다.
- 현재 운영 가정은 `VMF_IMPI` 또는 resolver 가 안정적으로 IMPI 를 제공하는 환경이다.
- iPhone 로그를 같이 수집하고 싶으면 각 명령 끝에 `--ios` 를 붙이면 된다.
- `NOTIFY`, `PUBLISH`, `REGISTER` 는 아직 `generator_complete` 이므로 기본 캠페인 셋에서 제외했다.

## 복붙용 세트

### 1. A31 INVITE baseline smoke - 실기기 MT INVITE 경로가 현재 환경에서 정상 울림과 응답을 내는지 가장 먼저 확인하는 무변이 스모크

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods INVITE --mt-invite-template a31 --ipsec-mode null --preserve-via --preserve-contact --layer wire --strategy identity --max-cases 3 --cooldown 1.0 --output a31-invite-smoke
```

### 2. A31 INVITE delivery-preserving campaign - 전달 가능성을 최대한 유지하면서 INVITE에서 얕지만 실제적인 변이를 넓게 훑는 기본 캠페인

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods INVITE --mt-invite-template a31 --ipsec-mode null --preserve-via --preserve-contact --profile delivery_preserving --layer model,wire,byte --strategy default --max-cases 30 --cooldown 1.0 --output a31-invite-delivery
```

### 3. A31 INVITE IMS-specific campaign - IMS/VoLTE 문맥에서 더 그럴듯한 필드와 헤더 교란을 노리는 INVITE 중심 캠페인

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods INVITE --mt-invite-template a31 --ipsec-mode null --preserve-via --preserve-contact --profile ims_specific --layer model,wire,byte --strategy default --max-cases 30 --cooldown 1.0 --output a31-invite-ims
```

### 4. A31 INVITE parser-breaker campaign - 파서 경계나 비정상 처리 경로를 더 세게 찌르는 INVITE 중심 공격 캠페인

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods INVITE --mt-invite-template a31 --ipsec-mode null --preserve-via --preserve-contact --profile parser_breaker --layer model,wire,byte --strategy default --max-cases 20 --cooldown 1.0 --output a31-invite-parser
```

### 5. Stateless runtime set: OPTIONS plus SUBSCRIBE - dialog setup 없이 바로 태울 수 있는 stateless runtime 메서드를 함께 훑는 확장 세트

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods OPTIONS,SUBSCRIBE --ipsec-mode null --profile delivery_preserving --layer model,wire,byte --strategy default --max-cases 30 --cooldown 0.7 --output a31-stateless-options-subscribe
```

### 6. Stateless MESSAGE set - pager-mode MESSAGE 처리와 body content 계열 변이를 비교적 안전하게 보는 단일 메서드 세트

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods MESSAGE --ipsec-mode null --profile delivery_preserving --layer model,wire,byte --strategy default --max-cases 20 --cooldown 1.0 --output a31-message-delivery
```

### 7. Invite-dialog core set: BYE plus INFO plus UPDATE plus REFER - INVITE로 상태를 세운 뒤 dialog 내부 메서드의 핵심 runtime path를 한 번에 보는 세트

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods BYE,INFO,UPDATE,REFER --ipsec-mode null --profile delivery_preserving --layer model,wire,byte --strategy default --max-cases 40 --cooldown 1.0 --output a31-dialog-core
```

### 8. PRACK focused set - reliable provisional response 전제의 PRACK 경로만 따로 보는 집중 세트

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods PRACK --ipsec-mode null --profile ims_specific --layer model,wire,byte --strategy default --max-cases 15 --cooldown 1.0 --output a31-prack-focus
```

### 9. CANCEL focused set - early-dialog 상태에서 CANCEL 처리와 teardown 성격을 보는 집중 세트

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods CANCEL --ipsec-mode null --profile delivery_preserving --layer model,wire,byte --strategy default --max-cases 15 --cooldown 1.0 --output a31-cancel-focus
```

### 10. INVITE wire edge set - wire 계층의 줄바꿈, whitespace, 중복 Content-Length, alias desync 같은 현실적인 edge-case만 따로 찌르는 세트

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods INVITE --mt-invite-template a31 --ipsec-mode null --preserve-via --preserve-contact --profile ims_specific --layer wire --strategy header_whitespace_noise,final_crlf_loss,duplicate_content_length_conflict,alias_port_desync --max-cases 24 --cooldown 1.0 --output a31-invite-wire-edges
```

### 11. INVITE byte edge set - tail chop, tail garbage, header-targeted byte 변이처럼 파서와 버퍼 경계에 가까운 바이트 변이를 모아보는 세트

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods INVITE --mt-invite-template a31 --ipsec-mode null --preserve-via --preserve-contact --profile parser_breaker --layer byte --strategy header_targeted,tail_chop_1,tail_garbage --max-cases 24 --cooldown 1.0 --output a31-invite-byte-edges
```

### 12. Dialog model breaker set - dialog 내부 메서드에 대해 model 계층의 state-breaking 성격 변이를 모아서 보는 세트

```bash
uv run fuzzer campaign run --mode real-ue-direct --target-msisdn 111111 --methods BYE,INFO,UPDATE,REFER,PRACK,CANCEL --ipsec-mode null --profile parser_breaker --layer model --strategy state_breaker --max-cases 24 --cooldown 1.0 --output a31-dialog-model-breaker
```

## 빠른 추천 순서

1. `1`번으로 환경과 baseline 확인
2. `2` 또는 `3`번으로 INVITE 기본 캠페인 실행
3. `7`, `8`, `9`번으로 dialog 계열 확장
4. `10`, `11`, `12`번으로 edge-case 중심 강화
